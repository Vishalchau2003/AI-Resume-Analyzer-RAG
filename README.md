# AI Resume Analyzer (RAG)

An AI-powered Resume Analyzer that compares resumes with job descriptions using Retrieval-Augmented Generation (RAG).

## Planned Features

- Resume Parsing
- JD Matching
- ATS Score
- Skill Gap Analysis
- Interview Question Generation
- Resume Improvement Suggestions
- Gemini API Integration
- FAISS Vector Database
- LangChain Pipeline

## Tech Stack

- Python
- LangChain
- FAISS
- Gemini API
- FastAPI
- Streamlit
